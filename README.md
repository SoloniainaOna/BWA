# BWA

I. BWA experiences
  - You can choose to use the fullscreen or not (suitable for every machine - computers, tablets, mobile phone).
  - Three experiences are available : Ripples & Trail effect ; Divide & Filters ; Puzzles.
  - In each experience, you can choose between 4 different videos, representing a new way to see the paintings.
  - WIP : Instagram integration, to post a feed on the BWA instagram of people's own art, with the divide and filters.

II. BWA experiences without ripples
  - Same as above, but without the Ripples & Trail experience.

III. BWA journey
  - Guide for the exhibition. Description of all the experiences.
  - Four steps :
    1. Roam in the room, watch from different angles, experiment all the interactive experiences
    2. Become an artist by altering the original paintings
